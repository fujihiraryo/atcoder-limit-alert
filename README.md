# 使い方
1. 適当なディレクトリに `git clone git@github.com:fujihiraryo/atcoder-limit-alert.git`
2. Chromeで `chrome://extensions/` にアクセス
3. デベロッパーモードをオンにする
4. 「パッケージ化されていない拡張機能を読み込む」でローカルにあるこのリポジトリを選択
